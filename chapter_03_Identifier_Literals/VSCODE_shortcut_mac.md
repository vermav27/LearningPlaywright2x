# VS Code Keyboard Shortcuts for macOS

| Shortcut | What it does |
|---|---|
| Cmd + P | Quick open a file |
| Cmd + Shift + P | Open Command Palette |
| Cmd + B | Toggle sidebar visibility |
| Ctrl + ` | Toggle integrated terminal |
| Cmd + S | Save file |
| Cmd + Shift + S | Save file as |
| Cmd + F | Find in file |
| Cmd + Option + F | Replace in file |
| Cmd + Shift + F | Find in workspace |
| Cmd + Shift + H | Replace in workspace |
| Cmd + / | Toggle line comment |
| Option + Shift + A | Toggle block comment |
| Cmd + D | Select next occurrence |
| Cmd + Shift + L | Select all occurrences |
| Cmd + Enter | Insert line below |
| Shift + Enter | Insert line above |
| Option + Up/Down | Move line up/down |
| Shift + Option + Up/Down | Copy line up/down |
| Cmd + Shift + K | Delete line |
| Cmd + ] / Cmd + [ | Indent/Outdent line |
| Cmd + Shift + \ | Jump to matching bracket |
| F12 | Go to definition |
| Option + F12 | Peek definition |
| Cmd + Shift + O | Go to symbol |
| Cmd + G | Go to line |
| Shift + Cmd + [ / ] | Navigate editors |
| Cmd + Shift + E | Focus Explorer |
| Cmd + Shift + G | Focus Source Control |
| Cmd + Shift + D | Focus Debug view |
| Cmd + Shift + X | Focus Extensions view |
| Cmd + Shift + V | Open Markdown preview |
| Cmd + K V | Open preview side by side |
| Cmd + Shift + N | New window |
| Cmd + W | Close editor |
| Cmd + Shift + W | Close window |
