# VS Code Keyboard Shortcuts for Windows

| Shortcut | What it does |
|---|---|
| Ctrl + P | Quick open a file |
| Ctrl + Shift + P | Open Command Palette |
| Ctrl + B | Toggle sidebar visibility |
| Ctrl + ` | Toggle integrated terminal |
| Ctrl + S | Save file |
| Ctrl + Shift + S | Save file as |
| Ctrl + F | Find in file |
| Ctrl + H | Replace in file |
| Ctrl + Shift + F | Find in workspace |
| Ctrl + Shift + H | Replace in workspace |
| Ctrl + / | Toggle line comment |
| Shift + Alt + A | Toggle block comment |
| Ctrl + D | Select next occurrence |
| Ctrl + Shift + L | Select all occurrences |
| Ctrl + Enter | Insert line below |
| Shift + Enter | Insert line above |
| Alt + Up/Down | Move line up/down |
| Shift + Alt + Up/Down | Copy line up/down |
| Ctrl + Shift + K | Delete line |
| Ctrl + ] / Ctrl + [ | Indent/Outdent line |
| Ctrl + Shift + \\ | Jump to matching bracket |
| F12 | Go to definition |
| Alt + F12 | Peek definition |
| Ctrl + Shift + O | Go to symbol |
| Ctrl + G | Go to line |
| Ctrl + Shift + Tab | Navigate editors |
| Ctrl + Shift + E | Focus Explorer |
| Ctrl + Shift + G | Focus Source Control |
| Ctrl + Shift + D | Focus Debug view |
| Ctrl + Shift + X | Focus Extensions view |
| Ctrl + Shift + V | Open Markdown preview |
| Ctrl + K V | Open preview side by side |
| Ctrl + Shift + N | New window |
| Ctrl + W | Close editor |
| Ctrl + Shift + W | Close window |
| Ctrl + Shift + P | Run command |
